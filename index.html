Voice Changer and Guitar Effects Unit
Created by Jonathan Lu and Ahsab Alebedin

Description

This project is a real-time voice and guitar signal processing unit. It captures continuous audio input via PCM (either from a microphone or directly from an electric guitar output) applies user-configurable digital effects, and plays the processed audio through a speaker in real time. The system supports dynamic control of effects and parameters, enabling live experimentation with audio transformations such as distortion, bitcrushing, and pitch shifting. Designed for embedded deployment, the device performs all sampling, processing, and playback using a CC3200 microcontroller, with minimal latency and no need for external computing resources.
